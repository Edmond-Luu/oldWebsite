Projects

On this page, you will find the web development projects that I have completed or am currently working on. All of my projects listed below are all hosted on my Github page.

The Tenzies game was made with React and was one of the final projects from the React course on Scrimba. However, I have added some extra features to this game to make it more enjoyable and user-friendly!
Click here to try the app!

The Measurement Converter is a large Javascript project that allows users to convert between imperial and metric measurements. I created this app because I find imperial measurements very confusing because, as I've said in my Fun Facts section, I use the metric units of measurement.
Click here to try the app!

The TV Show Search app is an app for users to type in a search query and it will display the cover art for TV shows matching the search terms. This project is derived from Colt Steele's Web Development Bootcamp, but I added new features to it and improved its aesthetics (with Bulma, a CSS framework). The first feature is a reset button, which resets the search input and clears all displayed cover arts. The second feature clears all displayed cover arts when the user enters a new search and clicks on the search button.
Click here to try the app!

The Language Translator app is a Javascript app that I've learned to create and it uses a free API translation service. This application has a feature in particular that allows users to adjust the speed of the voice output for the available languages. Such feature is very helpful for people learning a new language and want to improve their pronunciations.
Click here to try the app!

The Temperature Converter app is a simple app that allows the user to convert between fahrenheit and celsisus units. The app also displays units in Kelvin
Click here to try the app!
